== Description ==

Extra settings for Rocket.Chat's Wordpress plugin. Helps display better on sites that have WooCommerce activated, adjust if data is collected from signed in users and more.


== Installation ==

1. Upload the plugin files to the `/wp-content/plugins/rocketchat-extra-settings` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
